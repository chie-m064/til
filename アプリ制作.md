# アプリ制作

## 環境構築
### rbenvのインストール
複数のRubyバージョンを管理し、プロジェクト毎に異なるRubyのバージョンを使うことを実現してくれるツールです

```ruby
# 1) rbenvへのPATHを通すためのコマンド
$ echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc

# 2) rbenvを使うために必要な「rbenv init -」コマンドを設定するためのコマンド
$ echo 'eval "$(rbenv init -)"' >> ~/.bash_profile

# 3) 設定を反映するためのコマンド
$ source ~/.bash_profile
```
